Install Flutter on Windows

Steps to install Flutter SDK on Windows

  1. You go to the address, https://flutter.dev/docs/get-started/install/windows and download the latest version of Flutter SDK.
  2. Extract to any folder for example D:\flutter\
  3. Update system path for folder flutter\bin. In the Start search bar, type 'env' then select Edit environment variables for your account. Under User variables, check if you see the Path: box, then add the full path of the flutter\bin folder using use semicolon (;) to separate from other variables.
  4. Flutter provides a tool called "flutter doctor" to check all the necessary requirements for the Flutter development environment. Run "flutter doctor" using CMD utility.
  5. You run the command above so that the system checks and gives the following report Doctor summary (to see all details, run flutter doctor -v): [√] Flutter (Channel stable, v1.2.1, on Microsoft Windows [Version 10.0.17134.706], locale en-US) [√] Android toolchain - develop for Android devices (Android SDK version 28.0.3) [√] Android Studio (version 3.2) [√] VS Code, 64-bit edition (version 1.29.1) [!] Connected device ! No devices available ! Doctor found issues in 1 category. As mentioned above, we can see that Flutter SDK is installed, Android Tool is installed, Android Stuido is installed, there is no connection to the device, you need to connect the phone device via USB or turn on the virtual machine
  6. Install the latest Android SDK if you don't already have it installed and warned by flutter doctor
  7. Install latest Android Studio if you don't have it installed and warned
  8. Turn on android emulator or connect to an Android device
  9. Install Flutter and Dart plugins for Android Studio. These two plugins will provide templates for creating Flutter apps and options to run and debug Flutter apps on Android studio. Open Android Studio. Choose File → Settings → Plugins. Search for the Flutter plugin and click Install. Select Yes when the system asks to install Dart plugin. Restart Android studio.
